# 长篇修仙小说创作技能

## 任务目标

本 Skill 用于指导用户完成长篇修仙小说的系统化创作，采用"逐步完善"模式，从宏观设定逐步细化到微观章节。

## 修仙世界观基础

### 修为境界体系

**凡界阶段**：
- 炼气期（1-9层）：吸纳灵气入门
- 筑基期（初、中、后期）：凝聚道基
- 金丹期（初、中、后期、圆满）：丹成境界
- 元婴期（初、中、后期、圆满）：神魂出窍
- 化神期（初、中、后期、圆满）：神识化形
- 炼虚期（初、中、后期、圆满）：炼化虚妄
- 合道期（初、中、后期、圆满）：合道天地
- 大乘期（初、中、后期、圆满）：即将飞升
- 渡劫期：天雷淬体
- 飞升：进入仙界

**仙界阶段**：
- 地仙、真仙、金仙、太乙金仙、大罗金仙、准圣、圣人（混元大罗金仙）、天道圣人

### 世界架构

**三界**：人间界、幽冥界、仙界
**六道**：天道、人道、畜生道、饿鬼道、地狱道、阿修罗道
**势力划分**：正道宗门、魔道宗门、中立势力、散修联盟、皇室/王朝

### 核心设定要素

| 要素类别 | 可选设定 |
|---------|---------|
| 灵气类型 | 五行灵气、单一属性、变异属性（风雷冰毒等） |
| 功法等级 | 天地玄黄/一到九品/凡阶灵阶仙阶神阶 |
| 宝物等级 | 法器、灵器、宝器、法宝、灵宝、后天灵宝、先天灵宝 |
| 丹药等级 | 1-9品/下中上极 |
| 阵法等级 | 基础阵、困阵、杀阵、幻阵、复合阵 |
| 妖兽等级 | 1-9阶/低中高阶/凡兽灵兽神兽 |

## 创作阶段流程

本技能采用五阶段逐步完善创作法：

---

### 阶段一：世界观与核心设定

**创作内容**：
1. **中心思想**：小说要表达的核心主题
   - 例：逆天改命、与天争道、守护至亲、快意恩仇、红尘炼心

2. **世界观架构**：
   - 世界构成（三界/多界/单界）
   - 修炼体系（功法来源、境界设定）
   - 势力分布（宗门、皇室、散修）
   - 特殊规则（天道法则、灵气规则）

3. **贯穿全局角色完整设计**（阶段一必须完成）：
   - **主角**：姓名、外貌、性格、出身、家庭背景、人生经历、修炼资质、灵根属性、主修功法、核心动机、内心恐惧、行为底线、角色弧光（起点→转折→终点）
   - **核心配角**：师尊、挚友/兄弟、红颜/知己、劲敌/宿敌——每人必须设计：姓名、境界、身份、外貌、性格、行事风格、与主角关系、羁绊来源、人物弧光、结局
   - **主要反派**：阶段Boss、卷Boss、终极Boss——每人必须设计：姓名、境界、身份、外貌、性格、动机、野心、实力、弱点、布局、结局
   - **势力首领**：各势力宗主/皇帝等——姓名、境界、性格、势力、立场

4. **角色设计要求**：
   - 每个角色都要有独特的外貌特征（让人能记住）
   - 每个角色都要有鲜明的性格特点（避免脸谱化）
   - 每个角色都要有合理的动机（为什么这么做）
   - 每个角色都要有自己的故事（不是工具人）
   - 主角要有成长弧光，配角要有各自弧光，反派要有合理动机

5. **背景设定**：
   - 时代背景（太平盛世/乱世/复苏时代）
   - 地理环境（凡界版图、险地秘境）
   - 社会结构（宗门为尊/皇权至上/百家争鸣）

6. **最终目标**：
   - 主角最终达到的修为境界
   - 主角要完成的核心使命
   - 故事的核心悬念/谜底

**输出模板**：

```

### 一、中心思想
[核心主题，一句话概括]

- 世界构成：[描述]

- 势力分布：[描述]
```

### 阶段二：整体大纲规划

在阶段一确认后，创作整体故事大纲。

**创作内容**：

1. **主线设计**：主角修炼之路的关键转折点
   - 每个大境界的主要事件
   - 核心冲突的升级路径
   - 情感/羁绊的发展曲线

2. **辅线设计**：
   - 情感线（爱情、亲情、友情）
   - 宗门线（崛起、争斗、兴衰）
   - 势力线（正魔之战、朝堂风云）
   - 身世线（身世之谜、血海深仇）

3. **伏笔系统**：
   - 早期埋下的关键伏笔
   - 中期回收的伏笔
   - 贯穿全文的悬念

4. **高潮低谷节奏**：
   - 全书高潮点分布（至少3个大高潮）
   - 主角的低谷期设计
   - 节奏张弛设计

5. **角色发展线（完善）**：
   - 每个贯穿全局角色的出场时机
   - 每个角色的关键剧情节点
   - 角色关系的演变轨迹
   - 角色结局确认

### 阶段三：第一卷详细设计

在整体大纲确认后，开始第一卷的详细规划。

**创作内容**：

1. **卷目标**：
   - 本卷主角要达成的目标
   - 本卷要完成的核心事件
   - 本卷的情感基调

2. **卷背景**：
   - 时间线定位（第几年/什么时代节点）
   - 地点舞台（哪个区域/宗门/王朝）
   - 势力格局（本卷涉及的主要势力）

3. **卷主线**：
   - 核心冲突是什么
   - 主角如何行动
   - 关键转折点

4. **卷结构**：
   - 分成几个部分/大章节
   - 每个部分的核心任务
   - 高潮与缓冲安排

5. **伏笔埋设**：
   - 本卷需要埋下的伏笔
   - 与后续卷的关联
   - 需要回收的旧伏笔

**输出模板**：


### 阶段四：第一章细化创作

在第一卷设计确认后，开始第一章的详细创作规划。

**创作内容**：

1. **章目标**：
   - 本章主角要完成什么
   - 本章要传达什么信息
   - 本章的叙事功能（开篇/承上启下/高潮铺垫）

2. **章节结构**：
   - 起：如何开篇（直接冲突/场景描写/倒叙/悬念开场）
   - 承：事件展开
   - 转：冲突升级或转折
   - 合：本章结尾（留下悬念/完成小闭环）

3. **起伏设计**：
   - 本章是"起"还是"伏"
   - 若为"起"：设置什么期待点
   - 若为"伏"：埋设什么暗线
   - 如何为下一章/下一卷铺垫

4. **伏笔与细节**：
   - 本章需要埋下的伏笔
   - 需要刻画的细节（修炼体系、人物关系、场景氛围）
   - 需要展示的信息量控制

5. **视角与节奏**：
   - 主视角是谁
   - 信息披露节奏
   - 对话与描写的平衡

**输出模板**：

```
## 第四阶段输出：《第一章细化设计》

### 一、章目标
- **叙事功能**：[开篇/承上启下/冲突展开/高潮铺垫]
- **核心任务**：[描述]
- **信息传递**：[描述]

### 二、章节结构
| 结构 | 内容 | 篇幅占比 |
|-----|------|---------|
| 起（开篇） | [如何开场] | 约15% |
| 承（展开） | [事件发展] | 约40% |
| 转（转折） | [冲突升级/新信息] | 约30% |
| 合（结尾） | [悬念/小闭环] | 约15% |

### 三、起伏设计
- **本章定位**：[起/承/转/高潮/低谷]
- **期待点**：[读者会有什么期待]
- **铺垫任务**：[为后续章节埋设什么]

### 四、伏笔设计
| 伏笔 | 埋设位置 | 关联后续 |
|-----|---------|---------|
| ... | ... | ... |

### 五、角色安排
- **主视角**：[描述]
- **出场角色**：[列出]
- **重点刻画**：[角色关系/性格展示]

### 六、开篇方案
**推荐开篇方式**：[从以下选择或自创]
- 场景描写式：以某处场景入手，引出世界观和主角
- 直接冲突式：开篇即发生冲突，快速抓住读者
- 悬念式：以悬念/谜题开场，引发好奇
- 倒叙式：从结果倒推起点
- 日常切入式：从主角日常生活入手

[分场景/分段落写出本章大纲，包含关键对话、转折点]
```

## 使用示例

### 示例一：启动新小说创作
- 用户输入："我要写一个废材逆袭的修仙小说，主角是散修"
- 技能执行：引导进入阶段一，收集世界观和角色设定

### 示例二：继续已有创作
- 用户输入："第一卷设计好了，想继续第一卷的大纲"
- 技能执行：调用阶段四模板，引导细化第一章

### 示例三：特定环节咨询
- 用户输入："帮我设计一个强大的反派"
- 技能执行：提供反派设计方法论和模板

## 创作原则

### 角色设计原则

1. **贯穿全局角色必须完整设计**：主角、核心配角、主要反派、势力首领在阶段一就设计完整，不能边写边想
2. **每个角色都要饱满**：有独特外貌、鲜明性格、合理动机、自己的故事
3. **避免脸谱化**：正派不全善，反派不全恶
4. **临时角色按需设计**：在设计对应卷/章时才设计，每个临时角色也要有特点
5. **角色推动剧情**：让角色的选择推动故事，而非强行安排情节

### 创作流程原则

6. **循序渐进**：每个阶段确认后再进入下一阶段
7. **保持弹性**：大纲可以调整，不必墨守成规
8. **伏笔优先**：重要设定尽早埋设，为后续留足空间
9. **节奏把控**：高潮与低谷交替，避免单调
10. **详略得当**：重点章节重点写，过渡章节简洁处理

## 注意事项

- 修仙小说的核心是"成长感"，主角需要在每个阶段都有明确进步
- 反派要有合理的动机，避免脸谱化
- 情感线要自然发展，避免突兀
- 修炼体系要自洽，避免前后矛盾
- 世界观要服务于故事，不要过度堆砌设定
- 善用模板文档进行迭代创作，每次创作后更新对应文档
