## 六、常见问题清单

### 语句问题

| 问题 | 错误示例 | 正确示例 |
|-----|---------|---------|
| 句式单调 | 他跑了。他笑了。他哭了。 | 他撒腿就跑，笑得合不拢嘴，眼眶却红了。 |
| 形容词堆砌 | 非常极其特别极其 | 删掉，用动词替代 |
| 连续短句 | 好。很好。非常好。 | 调整节奏 |
| 主语缺失 | 看了看。想了想。 | 他看了看。他想了想。 |

### 描写问题

| 问题 | 错误示例 | 正确示例 |
|-----|---------|---------|
| 描写冗长 | 用300字描写一朵花 | 抓住特点，2-3句 |
| 感官单一 | 只有"看见" | 增加听觉、嗅觉等 |
| 比喻滥用 | 像什么一样像什么一样 | 精选比喻，宁缺毋滥 |
| 描写空洞 | 他很生气 | 他一拳砸在桌上，震得茶杯跳了起来 |

### 对话问题

| 问题 | 错误示例 | 正确示例 |
|-----|---------|---------|
| 对话书面化 | "我方才见此情景，心中甚是忧虑" | "我刚才看到了，吓死我了" |
| 对话标签化 | "你好"小明说。"再见"小红说。 | 用动作/表情替代"说" |
| 废话对话 | "你来了""是啊，我来了" | 删除或写出有效信息 |
| 信息灌输 | "我来告诉你，这个功法分九层..." | 融入剧情，自然传递 |

## 七、修改符号

### 常用标记
```
[删] 删除多余文字
[增] 添加必要文字
[换] 替换不当用词
[调] 调整语序
[分] 长句拆分
[连] 段落衔接
```

### 修改示例
```
原文：他的眼睛非常非常的大，而且特别的亮。
修改：他的眼眸又大又亮。

原文：战斗开始了他冲了上去然后他赢了。
修改：战斗开始。他如猛虎下山，一招制敌。

原文："你来啦""嗯，我来了""有什么事吗""没什么事"
修改："你来啦。""嗯。"两人对视，一切尽在不言中。
```

## 八、自检流程

### 写后三遍检查法

**第一遍：通读**
- [ ] 错别字
- [ ] 病句
- [ ] 标点

**第二遍：细读**
- [ ] 逻辑通顺
- [ ] 描写质量
- [ ] 对话自然

**第三遍：跳读**
- [ ] 节奏感
- [ ] 高潮点
- [ ] 记忆点

### 修改优先级

| 优先级 | 问题类型 |
|-------|---------|
| P0 | 错别字、病句、逻辑错误 |
| P1 | 重复描写、废话对话、节奏拖沓 |
| P2 | 文笔优化、细节打磨 |

- 节奏合适

- 有记忆点

### 优秀
- 文笔流畅
