# 角色设计模板

---

## 设计时机

### 贯穿全局角色（阶段一必须完成）

以下角色在阶段一（世界观与核心设定）时必须完整设计：

| 角色类型 | 必须设计的内容 |
|---------|--------------|
| **主角** | 全部字段：外貌、性格、出身、经历、修炼、动机、弧光、关系 |
| **核心配角** | 师尊、挚友、红颜、宿敌——每人必须完整设计 |
| **主要反派** | 阶段Boss、卷Boss、终极Boss——每人必须完整设计 |
| **势力首领** | 各势力宗主/皇帝——姓名、境界、性格、立场 |

### 临时角色（对应卷/章时按需设计）

以下角色在设计对应卷/章时才设计：

| 角色类型 | 设计时机 | 最低要求 |
|---------|---------|---------|
| **路人甲/杂鱼** | 出场时 | 姓名、身份、戏份 |
| **功能性角色** | 首次出场时 | 职业、性格、功能 |
| **宗门弟子** | 涉及宗门时 | 身份、阵营、关系 |
| **重要NPC** | 关键剧情前 | 完整设计 |
| **龙套模板** | 建立角色库时 | 基本信息即可 |

### 设计要求

每个角色都要做到：

1. **独特外貌**：让人能记住的特征（不是"普通"，而是具体）
2. **鲜明性格**：核心性格+优点+缺点
3. **合理动机**：为什么这么做
4. **独特故事**：不是工具人，有自己的背景
5. **成长弧光**：会变化、会成长（主角必备）

## 七、设计原则

1. **主角有缺陷才真实**
2. **配角各有功能，避免脸谱**
3. **反派有合理动机**
4. **临时角色不浪费笔墨**
5. **关系发展自然**
6. **实力体系自洽**
