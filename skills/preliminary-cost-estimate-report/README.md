# preliminary-cost-estimate-report

Produces a Class D (Schematic Design / Order-of-Magnitude) construction cost estimate report with an elemental cost breakdown, location adjustments, contingency tiers, and a clearly documented assumptions and exclusions log.

## When to Use

Use this skill when a quantity surveyor, cost manager, owner's representative, or project estimator needs to produce a preliminary construction cost estimate from limited project information (brief, concept design, or area schedule). Suitable for new construction, renovation, or civil works projects across any building type.

## What It Produces

- **DRAFT Class D Preliminary Cost Estimate Report** structured per AACE/RICS estimate classification
- **Elemental cost plan table** — 20 standard elements from demolition through site services, each with unit rate, quantity, total, and assumption
- **Location and market-condition adjustment** with stated index reference
- **Three-tier contingency build-up** — design, construction, and escalation contingency with stated rationale
- **Project cost additions** — professional fees, authority fees, and owner-supplied items for reference only
- **Assumptions and exclusions log** — numbered list of every assumption made
- **Open items list** — information gaps that must be resolved before upgrading to Class C
- **Cost manager review block** (unsigned) for QS verification before owner presentation

## Workflow Summary

1. **Intake** — project identification, type, GFA, construction system, occupancy, scope inclusions/exclusions, quality level, and available reference data
2. **Location Adjustment** — cost index region multiplier and market-conditions factor
3. **Elemental Build-Up** — cost per element using comparable data, published guides, or regional benchmarks
4. **Contingency Loading** — three tiers with stated rates and basis
5. **Report Assembly** — DRAFT Class D report with summary, cost rates, assumptions log, open items, and cost manager review block

## Scope and Boundaries

- Covers new construction, renovation, and civil works projects globally.
- Class D estimate only — not a tender price, GMP, or construction contract sum.
- Estimates become stale after 6 months without escalation reapplication.
- Unit rates are drawn from stated reference frameworks; cost manager must verify against current market data.
- Does not replace a full Bill of Quantities, trade-level takeoff, or formal tender process.
