# 止盈止损策略

## 核心原则

1. **先想亏损**：买入前先想好亏多少走人
2. **让利润奔跑**：不要过早止盈
3. **截断亏损**：不要死扛亏损
4. **分批操作**：不要一次性清仓

---

## 止盈策略

### 策略一：固定点位止盈

**适用**：短线、震荡市

**设置**：
| 涨幅 | 操作 |
|------|------|
| +10% | 减仓1/3 |
| +20% | 减仓1/3 |
| +30% | 清仓或留底仓 |

**优点**：简单明确
**缺点**：可能卖飞

### 策略二：移动止盈

**适用**：趋势行情、中长线

**设置**：
- 从最高点回撤 X% 触发止盈
- 常用回撤幅度：8-15%

**示例**：
```
买入价：$100
涨到$120，移动止盈线设$108 (回撤10%)
涨到$150，移动止盈线设$135
跌到$135触发止盈，锁定+35%利润
```

**优点**：吃到完整波段
**缺点**：可能回吐部分利润

### 策略三：技术位止盈

**适用**：技术派

**设置**：
- 压力位附近减仓
- 布林上轨附近减仓
- 前高附近减仓
- 缺口上沿减仓

### 策略四：估值止盈

**适用**：价值投资

**设置**：
| PE位置 | 操作 |
|--------|------|
| PE < 合理区间 | 持有/加仓 |
| PE = 合理区间 | 持有 |
| PE > 合理区间 × 1.2 | 减仓 |
| PE > 合理区间 × 1.5 | 清仓 |

### 策略五：时间止盈

**适用**：事件驱动、财报博弈

**设置**：
- 财报前减仓锁定利润
- 事件落地后离场
- 持有X个月后评估

## 止损策略

### 策略一：固定止损

**设置**：亏损 X% 无条件止损

| 风格 | 止损幅度 |
|------|----------|
| 超短线 | 3-5% |
| 短线 | 5-8% |
| 中线 | 8-15% |
| 长线 | 15-20% |

**注意**：
- 考虑波动率，高波动股止损放宽
- 避免止损设在整数位（容易被扫）

### 策略二：技术止损

**设置**：
- 跌破支撑位止损
- 跌破均线止损（MA20/MA50）
- 跌破前低止损
- 跌破趋势线止损

**技巧**：
- 支撑位下方2-3%设止损，避免假突破
- 成交量放大的跌破更可靠

### 策略三：基本面止损

**触发条件**：
- ❌ 财报miss（营收/EPS不及预期）
- ❌ 下调指引
- ❌ 核心业务恶化
- ❌ 行业逻辑破坏
- ❌ 管理层重大变动
- ❌ 监管/诉讼黑天鹅

**操作**：
- 基本面恶化 → 无条件离场
- 不死扛，不补仓

### 策略四：时间止损

**设置**：
- 持有X天不涨 → 离场
- 常用：5-20天

**适用**：短线、波段

## 仓位管理

### 凯利公式

**公式**：f = (p × b - q) / b

- f：仓位比例
- p：胜率
- q：败率 (1-p)
- b：盈亏比

**示例**：
- 胜率60%，盈亏比2:1
- f = (0.6 × 2 - 0.4) / 2 = 40%

**实际应用**：半凯利更安全，即20%

### 分批建仓

| 时机 | 仓位 |
|------|------|
| 初次建仓 | 30-50% |
| 确认突破 | 加仓20-30% |
| 回踩支撑 | 加仓20-30% |

### 分批止盈

| 涨幅 | 操作 |
|------|------|
| +15% | 减仓1/4 |
| +30% | 减仓1/4 |
| +50% | 减仓1/4 |
| 剩余 | 移动止盈 |

## 不同场景策略

### 场景一：短线博弈

**特征**：持有1-5天

**策略**：
- 止损：5-8%
- 止盈：10-20%，分2-3批
- 仓位：不超过总资金20%

### 场景二：波段操作

**特征**：持有1-3月

**策略**：
- 止损：10-15%
- 止盈：30-50%，移动止盈
- 仓位：不超过总资金30%

### 场景三：中长线投资

**特征**：持有6月-2年

**策略**：
- 止损：基本面止损为主
- 止盈：估值止盈为主
- 仓位：可满仓优质标的

### 场景四：财报博弈

**特征**：财报前后

**策略**：
- 财报前：减仓锁定利润
- 财报后：
  - 超预期 → 加仓/持有
  - miss → 无条件止损
- 仓位：财报前不超过30%

## 常见错误

### ❌ 错误做法

1. **不止损**：小亏变大亏
2. **频繁止损**：被来回打脸
3. **过早止盈**：卖飞大牛股
4. **死扛亏损**：越跌越补
5. **止损位太近**：正常波动就触发
6. **止损位太远**：亏损过大

### ✅ 正确做法

1. **买入前设止损**：先想亏多少
2. **执行纪律**：触发就执行，不犹豫
3. **复盘优化**：记录每次止盈止损，优化参数
4. **区分市场**：震荡市和趋势市策略不同
5. **情绪管理**：亏损后不报复性交易

## 快速决策树

```
买入后：
├─ 上涨
│   ├─ 涨幅<10% → 持有
│   ├─ 涨幅10-20% → 观察，考虑减仓
│   ├─ 涨幅>20% → 移动止盈，让利润奔跑
│   └─ 出现卖出信号 → 分批止盈
│
└─ 下跌
    ├─ 跌幅<止损线 → 观察
    ├─ 触发价格止损 → 执行
    ├─ 触发技术止损 → 执行
    ├─ 基本面恶化 → 无条件止损
    └─ 市场系统性风险 → 评估后决定
```
