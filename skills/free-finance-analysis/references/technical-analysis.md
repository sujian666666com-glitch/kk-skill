# 技术分析详解

## RSI (相对强弱指标)

**获取**: Benzinga直接提供

**判断标准**:
| RSI值 | 状态 | 操作含义 |
|-------|------|----------|
| > 80 | 极度超买 | 强烈卖出信号 |
| 70-80 | 超买 | 注意回调，可减仓 |
| 50-70 | 偏强 | 上涨趋势，持有 |
| 30-50 | 偏弱 | 下跌趋势，观望 |
| 20-30 | 超卖 | 可能反弹，可建仓 |
| < 20 | 极度超卖 | 强烈买入信号 |

**注意事项**:
- 强趋势中RSI可长期超买/超卖
- 背离：价格新高但RSI未新高 → 转折信号

---

## 支撑与压力

**识别方法**:
1. 52周高/低点（CNBC直接给）
2. 整数关口（如$100, $200, $300）
3. 前高/前低
4. 均线位置（MA20, MA50, MA200）

**强度判断**:
- 触及次数越多越强
- 时间跨度越长越强
- 成交量越大越强

**操作应用**:
- 支撑位附近 → 买入机会
- 压力位附近 → 卖出机会
- 跌破支撑 → 止损
- 突破压力 → 加仓

---

## 成交量分析

**量价关系**:
| 价格 | 成交量 | 含义 |
|------|--------|------|
| 上涨 | 放量 | 健康上涨 |
| 上涨 | 缩量 | 上涨乏力 |
| 下跌 | 放量 | 恐慌抛售，可能见底 |
| 下跌 | 缩量 | 阴跌，继续 |

**获取**: CNBC提供当日成交量，对比10日均量

---

## 缺口分析

| 缺口类型 | 特征 | 操作 |
|----------|------|------|
| 普通缺口 | 短期内回补 | 不重要 |
| 突破缺口 | 突破重要位置 | 追涨/追跌 |
| 中继缺口 | 上涨/下跌中途 | 继续持有 |
| 衰竭缺口 | 高位/低位放量 | 反转信号 |

---

## 趋势判断

**简单方法**:
1. 看价格 vs 52周高低点位置
2. 看近期涨跌幅
3. 看RSI在哪个区间

**趋势类型**:
| 特征 | 趋势 |
|------|------|
| 价格接近52周高点 + RSI>60 | 上涨趋势 |
| 价格接近52周低点 + RSI<40 | 下跌趋势 |
| 价格在52周中位 + RSI 40-60 | 震荡 |

---

## 综合判断流程

1. **看RSI** → 判断超买超卖
2. **看位置** → 价格在52周区间哪里
3. **看成交量** → 放量还是缩量
4. **看缺口** → 有无跳空
5. **多指标共振** → 不单一决策

**原则**:
- 技术面只是辅助，需结合基本面
- 不同市场有效性不同（美股>港股>A股）

---

## 注意：MACD/布林带/K线形态

这些指标需要：
1. 图表软件查看
2. 或自行计算（需要历史价格数据）

web_fetch只能获取当前数据，无法获取历史K线，所以：
- **能用**: RSI、支撑压力、成交量、缺口
- **不能直接用**: MACD、布林带、K线形态

如果需要这些，建议：
- 让用户自己看图表
- 或使用TradingView等工具
