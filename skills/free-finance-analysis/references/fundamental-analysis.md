# 基本面分析框架

## 快速评估清单

### 一句话判断
- PE < 行业平均 + 增速>15% + FCF为正 → 低估
- PE > 行业平均 × 2 + 增速<10% → 高估
- 营收连续下滑2季度 → 趋势恶化
- FCF连续为负 → 烧钱，谨慎

---

## 估值指标

### PE (市盈率)

**公式**: PE = 股价 / EPS

**判断**:
| PE值 | 相对判断 |
|------|----------|
| < 15 | 低估（传统行业） |
| 15-25 | 合理 |
| 25-40 | 偏高（成长股可接受） |
| > 40 | 高估（需高增速支撑） |
| 负数 | 亏损，不适用 |

**对比维度**:
1. vs 自身历史（5年/10年PE band）
2. vs 同行业可比公司
3. vs 增速（PEG = PE/增速，PEG<1低估）

**注意**:
- 周期股PE反着看（高PE买，低PE卖）
- 一次性收益需剔除
- 不同行业PE中枢不同

### PB (市净率)

**公式**: PB = 股价 / 每股净资产

**适用**: 银行、保险、地产等重资产行业

**判断**:
| PB值 | 含义 |
|------|------|
| < 1 | 低于净资产，可能低估 |
| 1-2 | 合理 |
| > 3 | 偏高 |

### PS (市销率)

**公式**: PS = 市值 / 营收

**适用**: 亏损的成长股、SaaS公司

**判断**: 
- PS < 3：低估
- PS 3-10：合理
- PS > 10：偏高

### PEG

**公式**: PEG = PE / 增速(%)

**判断**:
| PEG | 含义 |
|-----|------|
| < 1 | 低估 |
| 1-2 | 合理 |
| > 2 | 高估 |

## 成长性分析

### 营收增速

**判断标准**:
| 增速 | 评价 |
|------|------|
| > 30% | 高速成长 |
| 15-30% | 稳健成长 |
| 5-15% | 成熟期 |
| 0-5% | 停滞 |
| < 0 | 衰退 |

**关注**:
- 增速趋势：加速/稳定/放缓
- 同比vs环比：季节性影响
- 指引：管理层对未来增速预期

### 利润增速

**关键**:
- 利润增速 > 营收增速 → 规模效应/降本增效
- 利润增速 < 营收增速 → 成本上升/竞争加剧

### 分业务增速

拆分各业务线增速，判断：
- 哪个业务是增长引擎
- 哪个业务在拖后腿
- 新业务贡献占比

---

## 盈利能力

### 毛利率

**公式**: 毛利率 = (营收 - 成本) / 营收

**判断**:
| 毛利率 | 行业特征 |
|--------|----------|
| > 60% | 软件/服务/奢侈品 |
| 40-60% | 消费品/医药 |
| 20-40% | 制造业 |
| < 20% | 零售/大宗商品 |

**趋势**:
- 毛利率上升 → 定价权增强/成本下降
- 毛利率下降 → 竞争加剧/成本上升

### 净利率

**公式**: 净利率 = 净利润 / 营收

**判断**:
| 净利率 | 评价 |
|--------|------|
| > 20% | 优秀 |
| 10-20% | 良好 |
| 5-10% | 一般 |
| < 5% | 较差 |

### ROE (净资产收益率)

**公式**: ROE = 净利润 / 净资产

**判断**:
| ROE | 评价 |
|-----|------|
| > 20% | 优秀 |
| 15-20% | 良好 |
| 10-15% | 一般 |
| < 10% | 较差 |

**杜邦分解**:
ROE = 净利率 × 资产周转率 × 权益乘数
- 净利率：盈利能力
- 资产周转率：运营效率
- 权益乘数：杠杆水平

---

## 现金流分析

### 自由现金流 (FCF)

**公式**: FCF = 经营现金流 - 资本支出

**判断**:
| FCF | 含义 |
|-----|------|
| 正且增长 | 健康，可分红/回购 |
| 正但下降 | 增长放缓 |
| 负但收窄 | 投入期，关注拐点 |
| 负且扩大 | 烧钱，需关注现金储备 |

### 现金转换周期

经营现金流 / 净利润：
- > 1：利润质量高，现金实打实
- < 1：利润含应收账款/存货，质量存疑

### 资本支出

关注：
- 资本支出/营收比例
- 资本支出投向（扩张/维护）
- ROI（投入产出比）

---

## 财务健康

### 资产负债率

**公式**: 负债 / 资产

**判断**:
| 负债率 | 风险 |
|--------|------|
| < 30% | 低风险 |
| 30-50% | 适中 |
| 50-70% | 较高风险 |
| > 70% | 高风险 |

**行业差异**:
- 银行：高负债率正常
- 科技：低负债率常见
- 地产：高负债率需关注

### 利息覆盖倍数

**公式**: EBIT / 利息支出

**判断**:
- > 5：安全
- 3-5：正常
- < 3：危险

### 现金储备

关注：
- 现金及等价物
- 短期投资
- 是否足够覆盖短期债务

## 关键财报节点

### 季报关注点
1. 营收 vs 预期
2. EPS vs 预期
3. 指引（下季度/全年）
4. 电话会关键信息

### 年报关注点
1. 审计意见
2. 分业务详情
3. 风险因素
4. 管理层讨论与分析

### 财报日历
- 美股：每季度结束后4-6周
- A股：1季报4月底，半年报8月底，3季报10月底，年报4月底

## 红旗警示

出现以下情况需谨慎：
- ❌ 营收连续2季度下滑
- ❌ 应收账款增速 > 营收增速
- ❌ 存货增速 > 营收增速
- ❌ 经营现金流持续为负
- ❌ 审计意见非标准无保留
- ❌ 频繁更换审计机构
- ❌ 大额关联交易
- ❌ 管理层频繁减持
- ❌ 诉讼/监管风险

---

## 绿旗加分

出现以下情况可加分：
- ✅ FCF持续为正且增长
- ✅ 回购+分红
- ✅ 毛利率持续提升
- ✅ 新业务增速超预期
- ✅ 管理层增持
- ✅ 机构持仓增加

---

## 估值模型速查

### DCF (现金流折现)
适用于：成熟稳定公司
简化版：内在价值 = FCF × (1+增速) / (折现率-增速)

### PE估值
适用于：盈利稳定公司
合理股价 = EPS × 合理PE

### PS估值
适用于：成长股/亏损股
合理股价 = 营收 × 合理PS / 股本

### 分部估值
适用于：多业务公司
总价值 = Σ(各业务估值)
