# 缠论实战操作指南

> 从理论到实战：缠论买卖点速查、背驰判定标准、级别选择、操作纪律、常见错误。适用于A股日线/周线级别操作。

---

## 一、缠论核心概念速查

### 1.1 形态学基础概念

| 概念 | 定义 | 实战意义 |
|------|------|---------|
| **顶分型** | 三根K线中间最高、两端次高 | 潜在卖点区域 |
| **底分型** | 三根K线中间最低、两端次低 | 潜在买点区域 |
| **笔** | 顶底分型之间≥5根K线 | 走势的最小单元 |
| **线段** | 连续三笔构成 | 走势的方向单元 |
| **中枢** | 三段以上重叠区间 | 走势的"引力中心" |
| **走势类型** | 上涨/下跌/盘整 | 操作方向判断 |

### 1.2 动力学核心概念

| 概念 | 定义 | 实战意义 |
|------|------|---------|
| **背驰** | 创新高/低但力度减弱 | 趋势即将反转信号 |
| **趋势背驰** | 两个以上中枢的趋势中背驰 | 必返回最后一个中枢 |
| **盘整背驰** | 单一中枢震荡中的背驰 | 可能返回，需观察 |
| **MACD面积** | 红绿柱面积总和 | 量化背驰力度 |

### 1.3 三类买卖点速查

```
买点三兄弟（由下至上）：

第一类买点（抄底点）
  └─ 位置：下跌趋势背驰的最低点
  └─ 风险：最低（止损位明确）
  └─ 收益：最高（真正底部）
  └─ 频率：最少（需要趋势背驰）

第二类买点（确认点）
  └─ 位置：一买之后的向上回调低点
  └─ 风险：较低（已确认一买有效）
  └─ 收益：较高（趋势已启动）
  └─ 频率：中等

第三类买点（加速点）
  └─ 位置：向上离开中枢后的回调不破
  └─ 风险：中等（需观察是否有效突破）
  └─ 收益：较高（主升浪起点）
  └─ 频率：较多（中枢震荡中常见）

卖点对称：第三卖点→第二卖点→第一卖点
```

## 二、背驰判定标准

### 2.1 MACD辅助判定（最实用方法）

```
顶背驰判定流程：
  Step 1: 确认走势（两个以上中枢的上涨趋势）
  Step 2: 找到最近的离开段（从最后一个中枢向上离开的段）
  Step 3: 比较离开段与进入段的MACD面积
  Step 4: 离开段创新高但MACD面积/高度 < 进入段 → 顶背驰

量化标准：
  - 创新高（价格创阶段新高）
  - 但DIF值未创新高（相比进入段）
  - 或红柱面积未创新高
  - → 顶背驰成立

底背驰判定流程：
  - 下跌趋势中创新低
  - 但DIF值未创新低（相比进入段）
  - 或绿柱面积未创新低
  → 底背驰成立
```

### 2.2 均线辅助判定

| 方法 | 判定标准 | 适用场景 |
|------|---------|---------|
| MA5/MA10 | 价格创新高/低，但均线未跟随 | 辅助确认 |
| MA20/MA60 | 判断大级别方向 | 日线/周线 |
| 布林带 | 触及外轨后回归 | 超买超卖辅助 |

### 2.3 背驰强度评分

| 评分 | 背驰幅度 | 实战意义 |
|------|---------|---------|
| ★★★★★ | ≥50% | 极强，必返回中枢 |
| ★★★★ | 30-50% | 强，可重仓 |
| ★★★ | 20-30% | 中等，标准仓位 |
| ★★ | 10-20% | 弱，轻仓试探 |
| ★ | <10% | 很弱，观察为主 |

> 背驰幅度 = (进入段指标值 - 离开段指标值) / 进入段指标值

## 三、级别选择与操作匹配

### 3.1 级别与持仓周期对照

| 级别 | K线周期 | 适合人群 | 持仓周期 | 止损空间 |
|------|---------|---------|---------|---------|
| 最低级别 | 1分钟 | 短线高手 | 当日/隔日 | 1-2% |
| 分钟级 | 5-30分钟 | 有盯盘时间 | 1-7天 | 3-5% |
| **日线级** | 日K | **普通投资者（推荐）** | 1周-3月 | 5-10% |
| 周线级 | 周K | 长线投资者 | 1月-1年 | 10-20% |
| 月线级 | 月K | 超长线配置 | 1年以上 | 20%+ |

**缠论建议：日线级别是普通投资者的最佳起点**

### 3.2 多级别联立操作原则

```
大级别定方向，小级别定买点

月线级别（1个月以上）：
  └─ 判断：当前是牛市还是熊市？
  └─ 操作：月线向上笔 → 中线以上持有
  └─ 操作：月线向下笔 → 中线以上空仓/做空

周线级别（1周-3个月）：
  └─ 判断：在月线方向下的次级趋势
  └─ 操作：周线买点 → 买入
  └─ 操作：周线卖点 → 减仓/观望

日线级别（1周-3个月）：
  └─ 判断：精细化定位周线买卖点
  └─ 操作：日线一买/二买/三买 → 加仓/建仓
  └─ 操作：日线一卖/二卖/三卖 → 减仓/止损

30分钟/60分钟（1-7天）：
  └─ 用途：精确化入场点位（区间套）
  └─ 操作：找到日线买点对应的小级别入场点
```

## 四、止损与仓位管理

### 4.1 止损标准

| 买点类型 | 止损位 | 止损方式 |
|---------|--------|---------|
| 第一类买点 | 跌破第一买点价格3% | 止损出局 |
| 第二类买点 | 跌破第一类买点 | 止损出局 |
| 第三类买点 | 跌破中枢ZD（收盘确认） | 止损出局 |

### 4.2 仓位配置建议

```
缠论操作仓位原则：金字塔加仓法

第一买点（底背驰点）：
  └─ 初始仓位：20%（试探性建仓）
  └─ 止损：3%

第二买点（确认买点）：
  └─ 加仓：30%（确认一买有效后）
  └─ 累计仓位：50%
  └─ 止损：移动至成本价

第三类买点（加速买点）：
  └─ 加仓：30%（趋势确认启动）
  └─ 累计仓位：80%（留20%现金）
  └─ 止损：移动至第三买点入场价

总仓位：建议不超过80%，保留20%现金应对极端情况
```

### 4.3 出局原则

```
必须出局的信号：
  1. 跌破止损位（严格执行）
  2. 出现反向第一类买卖点
  3. 走势扩展为更大级别（中枢扩张）
  4. 大级别出现反向走势（月线笔改变方向）

可以持有的情况：
  1. 在中枢内震荡（正常持有）
  2. 向上离开中枢（继续持有）
  3. 回调不破第三类买点（加仓机会）
```

## 五、常见错误与避免方法

### 5.1 最高频的错误

| 错误 | 表现 | 解决方法 |
|------|------|---------|
| **在第一买点买入后频繁止损** | 买入后继续下跌，止损出局后又涨 | 第一买点前等待第二买点确认 |
| **把盘整背驰当趋势背驰** | 盘整中买入后继续震荡 | 确认有两个以上中枢才是趋势 |
| **小级别背驰大级别无引导** | 5分钟背驰但日线仍在下跌 | 严格遵守大级别引导原则 |
| **中枢扩张时不处理** | 中枢扩展后不知所措 | 按扩展后的中枢重新定位买卖点 |
| **重仓抄底** | 第一买点全仓买入 | 永远分批建仓，金字塔加仓 |

### 5.2 中枢操作常见误区

```
❌ 误区1：中枢内频繁高抛低吸
  问题：中枢震荡方向不确定，可能两边被打脸
  正确：在中枢内不操作，等待突破

❌ 误区2：在第三买点追高买入
  问题：第三买点后可能是假突破
  正确：第三买点轻仓，等待次级别回调确认

❌ 误区3：忽视大级别方向
  问题：日线买点买入，但周线仍在下跌
  正确：日线买点必须与周线方向一致

❌ 误区4：不设止损
  问题：走势可能长时间不返回中枢
  正确：任何买入必须有止损位

✅ 正确做法：多级别联立 + 分批建仓 + 严格止损
```

## 六、缠论实盘操作检查清单

每次操作前，逐项检查：

```
## ✅ 操作前检查清单（买入前必读）

### 大级别方向确认
- [ ] 月线方向：向上笔？向下笔？
- [ ] 周线方向：与月线同向？可以操作？
- [ ] 日线位置：当前处于哪一段？

### 买点有效性
- [ ] 是第几类买点？一买/二买/三买？
- [ ] 是否存在背驰？（MACD面积比较）
- [ ] 背驰幅度多少？（★评分）
- [ ] 小级别是否有引导？

### 风险控制
- [ ] 止损位在哪里？（明确到具体价格）
- [ ] 止损幅度多少？（≤10%）
- [ ] 计划仓位多少？（不超过80%）
- [ ] 是否金字塔建仓？

### 走势分析
- [ ] 当前有几个中枢？
- [ ] 中枢方向是什么？
- [ ] 距离止损位多远？

## ✅ 持仓中检查清单（每日/每周）

- [ ] 是否仍在中枢内震荡？
- [ ] 是否向上离开中枢？
- [ ] 是否出现顶背驰信号？
- [ ] 大级别方向改变了吗？
- [ ] 需要止损吗？

## ✅ 卖出后检查清单

- [ ] 是第几类卖点出的？
- [ ] 是否遵守了操作纪律？
- [ ] 盈亏情况如何？
- [ ] 总结经验教训
```
