## 三、贵州茅台经典案例（缠论第14课）

### 原始案例（2003-2008年）

```
缠论第14课核心：

贵州茅台（600519）周线级别分析：

【2003年】周线第一类买点
- 位置：约20元（不复权价格）
- 形成条件：
  1. 周线下跌趋势
  2. 两个周线下跌中枢
  3. 离开段与进入段背驰（MACD面积比较）
- 结果：这是缠论历史上的经典买点

【2007年】周线第一类卖点
- 位置：约200元+
- 形成条件：
  1. 周线上涨趋势
  2. 两个周线上涨中枢
  3. 趋势背驰
- 结果：之后茅台从200+跌至50元

【2008年】周线第二类买点
- 位置：约100元附近
- 形成条件：
  1. 不破2003年低点（约20元）
  2. 向上走势回调确认
- 结果：这是茅台历史上第二个绝佳买点
```

### 2020-2023年缠论案例

```
贵州茅台日线走势分解：

【2020年】日线第三类买点 + 主升浪
- 第三个日线中枢（1000-1300区间）形成
- 向上离开中枢：突破1300元
- 回踩确认：回调不破1000元（三买确认）
- 主升浪：1300→2600元

【2021年2月】日线第一类卖点
- 位置：约2600元
- 形成条件：
  1. 日线上涨趋势，两个以上中枢
  2. 离开段（2300→2600）vs 进入段（1700→2300）
  3. MACD背驰确认
- 结果：茅台从2600跌至1500元（跌幅42%）

【2022年】日线第二类卖点
- 位置：约1900-2000元
- 形成条件：
  1. 一卖后向上反弹
  2. 反弹不创新高（未破2600）
- 结果：再次下跌至1500元

【2023-2024年】日线中枢震荡
- 中枢区间：1500-1900元
- 走势类型：日线盘整（等待方向选择）
```

### 缠论启示（茅台案例总结）

```
从茅台案例中学到的：
1. 大级别买点（周线一买）= 超长期大机会
2. 三类买卖点按顺序出现（一买→二买→三买→一卖→二卖→三卖）
3. 趋势背驰后的反弹力度：完全返回最后一个中枢
4. 盘整走势：要等待方向选择，不要在中枢内频繁操作
5. 多级别联立：月线方向向上 → 周线买点才有效
```

## 三、选股策略

### 基于缠论的板块内选股
```
优先选择（在板块启动时）：
1. 板块内最先出现日线一买的个股
2. 走势与板块同步且力度更强的个股
3. 已有日线三买确认的个股

规避（在板块回调时）：
1. 板块下跌时率先破位的个股
2. 力度弱于板块平均水平的个股
3. 中枢扩张/扩展的个股
```
