# Landscape Site Analysis Drafter

Converts site data, field observations, and desktop research into a DRAFT landscape site analysis report that feeds the design development phase. Covers physical inventory, microclimate, visual and contextual analysis, regulatory constraints, and an opportunities-and-constraints synthesis with design program recommendations.

## Use Cases

- Registered landscape architects (RLA/ASLA/CSLA) preparing schematic-phase documentation
- Site designers documenting pre-design investigations for client presentations
- Planning consultants synthesizing site data for feasibility studies or master plans
- Landscape architecture interns compiling field notes into structured analysis reports

## Scope

**In scope:**
- Physical site inventory: topography, soils and geology, hydrology and drainage, existing vegetation
- Microclimate analysis: solar access, prevailing winds, USDA plant hardiness zone, frost dates
- Visual and contextual analysis: views, adjacencies, noise/odor sources, cultural and historic features
- Regulatory inventory: zoning setbacks, easements, wetland buffers, ADA / accessibility requirements, utility corridors
- Opportunities-and-constraints matrix
- Design program recommendations based on analysis findings

**Out of scope:**
- Wetland delineations and jurisdictional determinations — require a licensed wetland scientist or environmental professional
- Geotechnical investigations — require a licensed geotechnical engineer
- Structural calculations for retaining walls, grading, or slope stability
- Final permit applications or regulatory submissions
- Landscape construction documents or planting specifications

## Output

A DRAFT site analysis report containing:
- Project identification and scope summary
- Physical inventory narrative and data tables
- Microclimate summary
- Visual and contextual analysis narrative
- Regulatory constraint inventory
- Opportunities-and-constraints matrix (tabular and narrative)
- Design program recommendations
- Data-gap list with field-verification flags
- RLA review and stamp block

**This document is a DRAFT only.** A registered landscape architect must review the analysis, verify field conditions, confirm regulatory requirements with the AHJ, and seal the document before it is presented to clients or submitted to any authority.

## Safety and Boundaries

- Always note data gaps that require physical field verification before design decisions are made.
- Critical environmental features (wetlands, floodplains, protected species habitat, cultural resources) require licensed environmental, biological, or cultural-resource professionals — this skill flags them but does not evaluate them.
- Regulatory requirements vary by jurisdiction; always direct the LA to verify setbacks, buffers, and easement restrictions with the AHJ and a title report.
