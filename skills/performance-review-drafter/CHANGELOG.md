# Changelog

## [0.1.1] - 2026-05-19
Renamed skill slug from `performance-review-writer` to `performance-review-drafter` to resolve a publish-time slug conflict.

## [0.1.0] - 2026-05-19
Initial release. Guides managers through structured evidence gathering, five-section review drafting (Summary, Goals & Achievements, Core Competencies, Development Areas, Next Period Goals), and an automated bias check covering recency bias, halo/horn effects, vague language, attribution errors, and protected characteristics.
