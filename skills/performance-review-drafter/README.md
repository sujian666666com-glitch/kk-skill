# Performance Review Drafter

**Domain:** HR · People Operations
**Platforms:** Claude · Codex

## Purpose

Helps managers draft complete, evidence-based, bias-checked annual or mid-year performance reviews for direct reports. Turns raw notes, goals data, and observations into a structured, manager-ready document.

## When to Use

- When review season arrives and you need to write a performance review for a direct report
- When you have notes and observations but need to organize them into a structured document
- When you want a built-in bias check before submitting a review
- When you need help drafting specific, measurable goals for the next review period

## What It Does

1. Collects goals, accomplishments, feedback, and competency data through structured questions — one at a time
2. Confirms the evidence inventory with the manager before drafting anything
3. Drafts all five standard review sections: Summary, Goals & Achievements, Core Competencies, Development Areas, and Next Period Goals
4. Runs a built-in bias check covering recency bias, halo/horn effects, vague language, attribution errors, and protected characteristics
5. Delivers a clearly labeled draft with a self-review note and an invitation to refine

## Notes

This skill produces a **draft** — the manager must review and finalize before submitting. All accomplishments, outcomes, and ratings must come from the manager; the skill never invents performance data.

For reviews at companies with specific legal or regulatory requirements (e.g., unionized workplaces, government roles), the manager should have HR or legal review the final text.
