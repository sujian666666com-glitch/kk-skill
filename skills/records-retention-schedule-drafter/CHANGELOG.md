# Changelog

## [0.1.0] - 2026-05-25
Initial release. Adds an ARMA GARP / IGIM-aligned records-retention-schedule drafting workflow that captures scoped intake (organization type, sector, jurisdictions, regulators, lines of business, existing schedule and gaps, system landscape, retention-program owner), selects schedule structure (big-bucket / functional / hybrid / hierarchical), inventories record series with a controlled vocabulary (function → activity → series → sub-series), captures the four-pillar retention basis (statutory / regulatory / operational / historical-or-cultural) with named citations, defines event-driven triggers (active period + retention period + final action), addresses format-specific disposition (paper / electronic / email / chat-and-collaboration / structured-data / backup-and-DR / physical media), drafts the legal-hold and litigation-hold-suspension protocol, applies privacy-law minimization overlays (GDPR Art 5(1)(e), CCPA / CPRA storage limitation, HIPAA, FERPA, GLBA, PCI-DSS, PIPEDA, LGPD, APPI, state and provincial laws), runs a defensible-disposition workflow (review → approval → execution → audit trail), runs a GARP / IGIM and ARMA principles self-check, maintains a basis-of-retention register, and produces a DRAFT retention schedule with an unsigned Records-Officer / Privacy-Officer / General-Counsel sign-off block — for governance-committee review, never an executed records-control policy, never legal advice.
