# 学术写作要点

> 学术论文、研究报告的英文写作规范。

---

## 核心原则

### 正式度
- 避免缩写：do not, cannot, it is（不用 don't, can't, it's）
- 避免口语化表达：a lot of → substantial / considerable
- 避免感叹号和缩写

### 客观性
- 减少第一人称（I, we），除非说明研究方法
- 用 "The data suggest..." 替代 "I think the data shows..."
- 用 "Evidence indicates..." 替代 "We believe..."

### 精确性
- 用具体数据替代模糊描述：
  - "a significant number" → "73% (n=1,303)"
  - "recently" → "in 2025" / "over the past 12 months"
  - "many" → "15 of 20 participants (75%)"

### 谨慎表达（Hedging）
学术写作需要适度保留，但不过度：
- ✅ "These findings suggest that..."
- ✅ "The evidence points to..."
- ❌ "This proves beyond doubt that..."
- ❌ "I think maybe it could potentially be..."

## 段落结构

### PEEL 结构
- **P**oint：段落核心论点（首句）
- **E**vidence：数据、引用、案例支撑
- **E**xplanation：分析与阐释
- **L**ink：与前后段落的连接

### 连接词
| 功能 | 推荐 | 避免 |
|------|------|------|
| 递进 | Furthermore / Moreover / In addition | Also / And also |
| 转折 | However / Nevertheless / Conversely | But / On the contrary |
| 因果 | Consequently / As a result / Therefore | So / Because of this |
| 举例 | For instance / To illustrate | Like / Such as（句首） |
| 总结 | In summary / To conclude | In a word / All in all |

## 常见学术表达错误

| 错误 | 正确 |
|------|------|
| a research | research（不可数） |
| researches | research / studies |
| an advice | advice（不可数） / a piece of advice |
| informations | information（不可数） |
| phenomenons | phenomena |
| criterias | criteria（criteria 已是复数） |
| the data shows | the data show（data 学术用法中为复数） |
| compare with/to | compare with（差异）/ compare to（相似） |
| different than | different from |
