# 常见中式英语模式（Chinglish Patterns）

> 中国作者写英文材料时最常出现的翻译腔和中式英语模式。审查时重点关注这些高频错误。

---

## 介词滥用

| 错误表达 | 正确表达 | 说明 |
|----------|----------|------|
| discuss about X | discuss X | discuss 是及物动词 |
| explain about X | explain X | explain 及物 |
| emphasize on X | emphasize X | emphasize 及物 |
| contact with X | contact X | contact 作动词时及物 |
| affect on X | affect X | affect 及物 |
| similar like X | similar to X | similar + to |
| compare to X (主动) | be compared to X | 被动更常见 |
| according to me | in my opinion | according to 用于引用他人话 |
| due to this reason | for this reason | due to 修饰名词，不独立使用 |

## 冠词问题

| 错误表达 | 正确表达 | 说明 |
|----------|----------|------|
| go to company | go to the company | 具体公司需要冠词 |
| as manager | as the manager | 特定职位需要冠词 |
| make decision | make a decision / make decisions | 可数名词 |
| with pleasure | with great pleasure / gladly | 固定搭配不完整 |
| in market | in the market | 需要冠词 |

## 中式直译词组

| 中式表达 | 母语习惯 |
|----------|----------|
| very very good | excellent / outstanding |
| make our best effort | do our best |
| face many challenges | encounter numerous challenges |
| under the leadership of | led by / guided by |
| with the development of | as X develops / with the rise of |
| in the process of | while / as |
| according to the actual situation | depending on the circumstances |
| strengthen the management of | improve management of |
| promote the development of | advance / foster / drive |
| play an important role | play a key role / be instrumental |
| lay a solid foundation | establish a strong foundation |
| in this regard | in this respect / here |
| at the same time | meanwhile / simultaneously |
| on the one hand... on the other hand | 仅用于真正对立的观点 |
| generally speaking | 通常删掉 |

## 句子结构问题

| 中式习惯 | 母语习惯 |
|----------|----------|
| There are many people who believe that... | Many people believe that... |
| It is well known that... | 直接陈述事实，或用 Evidently / Clearly |
| In today's society, with the rapid development... | 删掉或简化为 As X grows... |
| This shows that... (句首) | This suggests / indicates / demonstrates |
| From this we can see that... | This shows / reveals |
| Based on the above analysis, we can conclude... | In conclusion / Therefore |
| Only in this way can we... | This approach enables us to... |
| Not only... but also... (滥用) | 直接列出，不强调关联 |
| It is worth mentioning that... | Notably / Importantly / 直接陈述 |
| As far as I am concerned | I believe / In my view |

## 冗余表达

| 冗余表达 | 简洁版本 |
|----------|----------|
| in order to | to |
| at this point in time | now |
| due to the fact that | because |
| for the purpose of | to |
| with regard to / in regard to | about / regarding |
| at the present time | now |
| in the near future | soon |
| a large number of | many |
| a great deal of | much |
| it goes without saying that | 删掉 |
| need to further optimize | need to optimize |
| actual implementation | implementation |
| past history | history |
| end result | result |
| true fact | fact |

## 标点与格式

| 问题 | 说明 |
|------|------|
| 中英文标点混用（，。""） | 全部改为英文标点 |
| 句号后空一格（中文习惯不空格） | 英文句号后必须空一格 |
| 括号紧贴前词（中文习惯） | 英文括号前空一格 |
| 破折号用 —— | 英文用 — 或 -- |
| 引号用 "" | 英文用 "" 或 '' |
| 省略号用 …… | 英文用 ... |
| 书名号《》 | 英文用斜体或引号 |
