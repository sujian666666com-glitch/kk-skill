# 商务写作风格指南

> 商务场景（邮件、报告、演示文稿、提案）的英文写作风格规范。

---

## 核心原则

### 简洁优先
- 一句话一个核心观点
- 能删就删：very, really, basically, actually, in order to
- 商务写作目标：让读者在 5 秒内理解你要说什么

### 主动语态优先
| 被动 | 主动 |
|------|------|
| A decision was made | We decided |
| It has been determined | Our analysis shows |
| Steps will be taken | We will take steps |
| It is recommended that | We recommend |

### 直接表达
| 委婉 | 直接 |
|------|------|
| I think we might want to consider... | I recommend... |
| It might be a good idea to... | Let's... |
| We would like to inform you that... | [直接说事] |
| Please be advised that... | [直接说事] |

## 邮件写作

### 主题行
- 明确具体：Update on Q3 Revenue Analysis
- 含行动项：Action Required: Submit Q4 Budget by Friday
- 避免模糊：Important / FYI / Hello

### 开头
- 商务邮件不需要 Dear X, I hope this email finds you well
- 直接进入主题：I'm writing to share... / Following up on...
- 回复邮件不需要重新寒暄

### 正文
- 使用项目符号列出不超过 5 点的要点
- 每点一行，动词开头
- 行动项加粗并明确截止日期

### 结尾
- 用 Next steps 替代 Please let me know if you have any questions
- 明确谁在什么时候做什么

## 报告写作

### 结构
1. Executive Summary（1 段，≤ 150 词）
2. Background / Context
3. Analysis / Findings
4. Recommendations
5. Next Steps

### 数据呈现
- 数字后单位紧跟：$1.2M, not 1.2 million dollars
- 百分比用数字+符号：23%, not twenty-three percent
- 大数字用缩写：1.2M, 45K（首次出现可标注全称）

### 语气
- 用数据支撑论点，不用形容词堆砌
- "significant increase of 23%" 优于 "very big increase"
- "declined by 5%" 优于 "went down a little bit"
