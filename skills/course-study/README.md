# course-study

> 把一门课变成全覆盖、费曼式讲解、可应试的复习笔记 —— 一个点都不漏。

[English](README.en.md) · **简体中文**

**做什么** —— 把一门课的材料（PDF 讲义、粘贴的提纲，或仅一个课程名）变成全覆盖、费曼式讲解、可应试的复习笔记。

**好在哪** ——
- **强制完整性**：先生成覆盖清单，再逐条对账 —— 任何抽取到的知识点都不会被悄悄漏掉。
- **每个概念都有费曼块**：白话 → 直觉 → 形式 → 例题 → 误区；从不上来就甩形式定义。
- **刻意精简**：砍掉了刷题、间隔重复、Anki 导出，只做一个干净的复习工具。

**什么时候用** —— 「复习这门课 / 整理这份讲义」·「给我的期中/期末/小测做份复习提纲 / cheat sheet」·「把 X 这个知识点讲简单点」；也可用 `/course-study` 显式调用。
**不适用** —— 专辑 / 乐评（→ album-review）；开放式的辅导对话；替你解 / 做要提交的作业或考题；既无材料、又找不到标准教学大纲的课程。

**安装** —— `npx skills add VincentJiang06/skills`（或 `cp -R skills/course-study ~/.claude/skills/`）。

完整说明见 [SKILL.md](SKILL.md)。
