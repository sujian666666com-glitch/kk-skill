## Description: <br>
Converts user-provided UI screenshots or images into runnable HTML/CSS, React/Tailwind components, or layout descriptions. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and designers use this skill to turn UI screenshots, mockups, or pasted images into working frontend code, or to get a layout description before implementation. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Screenshots can contain secrets, customer data, internal dashboards, personal information, or confidential designs. <br>
Mitigation: Redact sensitive areas before providing images to the skill, and review generated code before deployment. <br>
Risk: Generated UI code is an approximation and may include inaccurate layout, colors, spacing, or placeholder content. <br>
Mitigation: Review the output against the original image and fine-tune visual details before using it in production. <br>
