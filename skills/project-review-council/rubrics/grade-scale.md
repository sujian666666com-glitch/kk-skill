# 评级换算（A+ ~ F）

## 综合分 → 评级映射

基于 P7 商业评估的加权综合分（0~10）。

| 综合分 | 评级 | 含义 |
|--------|------|------|
| ≥ 9.0 | **A+** | 卓越，强烈建议加速 |
| 8.0 - 8.9 | **A** | 优秀，建议继续 |
| 7.0 - 7.9 | **B+** | 良好，建议继续 |
| 6.0 - 6.9 | **B** | 及格，谨慎继续 |
| 4.0 - 5.9 | **C** | 不足，需重大改进 |
| 2.0 - 3.9 | **D** | 危险，建议暂停 |
| < 2.0 | **F** | 失败，建议放弃 |

## 修正规则

评级可被以下因素修正（不超过 ±1 档）：

| 因素 | 修正 |
|------|------|
| 存在 Fatal 风险 | 降 1 档 |
| 壁垒评分（P7）≥8 且均分≥7 | 升 1 档（A+ 封顶） |
| P10 Red Team 成立论据显著占优 | 降 1 档 |
| P10 Blue Team 成立论据显著占优 | 升 1 档（A+ 封顶） |

## 评级与建议的对应

| 评级 | 默认建议 |
|------|---------|
| A+ / A | 继续（加速） |
| B+ / B | 继续 |
| C | 暂停（改进后重评） |
| D | 重构 / 暂停 |
| F | 放弃 |

## 反 Gaming

- 禁止只看均分：必须结合 P5 风险等级综合判断
- 壁垒评分权重高（0.15），防止"高增长低壁垒"项目虚高
- Devil's Advocate 与 Competitor 的评分拉低均分是设计意图，不可剔除
