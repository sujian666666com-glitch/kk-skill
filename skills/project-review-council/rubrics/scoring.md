# 评分标准（0~10）

## 分段定义

| 分段 | 含义 | 判定锚点 |
|------|------|---------|
| **9-10** | 卓越 | 有证据 + 行业领先 + 可复制壁垒 + 创新突破 |
| **7-8** | 良好 | 基本完整 + 少量待验证 + 有差异化 |
| **5-6** | 及格 | 多处假设 + 需补证据 + 无明显壁垒 |
| **3-4** | 不足 | 逻辑漏洞 / 关键缺失 / 严重同质化 |
| **0-2** | 危险 | 自相矛盾 / 致命缺陷 / 无价值 |

## 防 Gaming 规则

1. **证据强制** — 评分必须附 ≥1 条证据或明确标注【推测】
2. **Auditor 否决权** — Independent Auditor 可驳回无证据的高分（≥8）
3. **禁止趋同** — 角色评分极差 <2 触发独立性审查（疑似互相迎合）
4. **Devil's Advocate 约束** — 除非有压倒性证据，否则评分 ≤4
5. **整分禁用** — 禁止只给 5/10 这类整十分，必须细化（如 7/10）

## 评分维度适用

- P3 单角色评分：对该角色视角的项目评分
- P7 商业评估 11 维：每维独立评分
- P9 综合评级：由 P7 加权综合分映射到 [grade-scale.md](./grade-scale.md)
